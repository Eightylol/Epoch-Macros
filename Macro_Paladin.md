# Paladin Macros

### Crusader Strike
```
#showtooltip Crusader Strike
/startattack
/cast Crusader Strike
```

### Divine Shield
```
#showtooltip
/cancelaura Divine Shield
/cast Divine Shield
```

### Judgement
```
#showtooltip Judgement
/startattack
/use 14
/cast Judgement
/cast Seal of Command(Rank 1)
```

### Seal of Command
```
#showtooltip Seal of Command
/startattack
/cast Seal of Command
```
