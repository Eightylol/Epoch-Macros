# Hunter Macros

### Misdirection
```
#showtooltip Misdirection
/cast [@focus,exists][@pet] Misdirection
```
### Hunter's Mark /w Skull
```
#showtooltip Hunter's Mark
/script SetRaidTarget("target",8);
/cast Hunter's Mark
```

### Kill Command
```
#showtooltip Kill Command
```
