# Druid Macros

### Bash
```
#showtooltip Bash
/cast !Bear Form
/cast Bash
```

### Cat
```
#showtooltip
/cast !Cat Form
/startattack
```

### Challenging Roar
```
#showtooltip Challenging Roar

```

### Travel

#showtooltip
